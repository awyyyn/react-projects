# ReFitness

Refitness provides you the differnt types of fitness exercise.

### Tools 
- Vite + ReactJS
- TypeScript
- Framer Motion
- Tailwind CSS
- Rapid API

### Run in local machine
```git

  // clone repo
  git clone https://github.com/awyyyn/react-projects.git
  cd react-projects/fitness-tracker

  // install dependencies
  npm install

  // run locally
  npm run dev

  // build and deploy
  npm run build

  // preview build
  npm run preview
```
